# 第一章：人工智能基础概论

人工智能（Artificial Intelligence，AI）是计算机科学的一个分支，旨在创建能够模拟人类智能行为的系统。这些系统能够学习、推理、感知环境并做出决策。

## 1.1 人工智能的定义与历史

人工智能的概念最早由约翰·麦卡锡在1956年的达特茅斯会议上提出。会议上，科学家们讨论了如何让机器模拟人类智能的各个方面的可能性。

从1956年至今，人工智能经历了三次发展浪潮：
- 第一次浪潮（1956-1974）：符号主义AI，基于逻辑推理和专家系统
- 第二次浪潮（1980-1987）：知识工程时代，专家系统商业化
- 第三次浪潮（2012至今）：深度学习革命，大数据驱动

## 1.2 机器学习的基本概念

机器学习是人工智能的一个核心子领域。它使计算机系统能够从数据中自动学习规律，而无需显式编程。

机器学习主要分为三种类型：

### 1.2.1 监督学习

监督学习是最常见的机器学习范式。在监督学习中，算法使用带标签的训练数据来学习输入与输出之间的映射关系。

常见的监督学习算法包括：
- 线性回归（Linear Regression）
- 逻辑回归（Logistic Regression）
- 支持向量机（Support Vector Machine）
- 决策树（Decision Tree）
- 随机森林（Random Forest）

### 1.2.2 无监督学习

无监督学习处理未标注的数据，目标是发现数据中隐藏的结构和模式。

主要方法包括：
- 聚类算法：K-means、DBSCAN、层次聚类
- 降维技术：主成分分析（PCA）、t-SNE

### 1.2.3 强化学习

强化学习通过与环境交互来学习最优策略。智能体在环境中采取行动，根据获得的奖励信号来调整策略。

## 1.3 神经网络与深度学习

神经网络是受人脑结构启发的计算模型。深度学习是机器学习的一个子领域，它使用多层神经网络（深层神经网络）来学习数据的层次化表示。

### 1.3.1 感知机与多层感知机

感知机是最简单的神经网络单元，由罗森布拉特在1958年提出。多层感知机（MLP）通过堆叠多个感知机层，能够学习非线性决策边界。

### 1.3.2 卷积神经网络

卷积神经网络（CNN）是处理网格状数据（如图像）的专用神经网络架构。它通过卷积层提取局部特征，池化层降低维度。

经典CNN架构：
- LeNet-5（1998）
- AlexNet（2012）
- VGGNet（2014）
- ResNet（2015）

### 1.3.3 循环神经网络与LSTM

循环神经网络（RNN）擅长处理序列数据。LSTM（长短期记忆网络）通过门控机制解决了传统RNN的梯度消失问题。

## 1.4 深度学习框架

现代深度学习依赖于高效的计算框架：

| 框架 | 开发者 | 特点 |
|------|--------|------|
| TensorFlow | Google | 生态完善，生产部署友好 |
| PyTorch | Meta | 动态图，研究友好 |
| JAX | Google | 函数式编程，自动微分 |
| PaddlePaddle | 百度 | 中文文档丰富 |

## 1.5 人工智能的应用领域

人工智能已渗透到各行各业：

**计算机视觉**：
- 图像分类与识别
- 目标检测与跟踪
- 图像分割与生成

**自然语言处理**：
- 机器翻译
- 文本分类与情感分析
- 问答系统与对话机器人

**语音识别**：
- 语音转文字
- 声纹识别
- 语音合成

**推荐系统**：
- 协同过滤
- 基于内容的推荐
- 深度学习推荐模型

## 1.6 人工智能的伦理与挑战

随着AI技术的快速发展，也带来了一系列伦理和社会挑战：

1. **数据隐私**：AI系统需要大量数据训练，如何保护个人隐私？
2. **算法偏见**：训练数据中的偏见可能被模型学习和放大
3. **就业影响**：自动化可能替代部分人工岗位
4. **安全风险**：深度伪造、自主武器等技术的滥用

# 第二章：Transformer架构与大语言模型

## 2.1 注意力机制

注意力机制是Transformer架构的核心创新。它允许模型在处理序列时，动态地关注输入的不同部分。

自注意力（Self-Attention）的计算公式：

Attention(Q, K, V) = softmax(QK^T / sqrt(d_k)) V

其中Q、K、V分别是查询、键、值矩阵，d_k是键向量的维度。

## 2.2 Transformer架构详解

Transformer由编码器（Encoder）和解码器（Decoder）组成。编码器将输入序列映射为连续表示，解码器根据编码器输出生成目标序列。

### 2.2.1 位置编码

由于Transformer没有循环结构，需要显式编码位置信息。原始Transformer使用正弦/余弦位置编码：

PE(pos, 2i) = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))

### 2.2.2 多头注意力

多头注意力将查询、键、值投影到多个子空间，并行计算注意力，然后将结果拼接：

MultiHead(Q, K, V) = Concat(head_1, ..., head_h) W^O

## 2.3 预训练与微调

大语言模型通常采用两阶段训练策略：

1. **预训练**：在大规模无标注语料上进行自监督学习
   - 掩码语言建模（MLM）
   - 因果语言建模（CLM）

2. **微调**：在特定任务的标注数据上进行监督学习
   - 全量微调
   - 参数高效微调（LoRA、Adapter、Prompt Tuning）

## 2.4 大语言模型的涌现能力

当模型规模超过一定阈值时，会涌现出训练时未明确设计的能力：

- 上下文学习（In-context Learning）
- 思维链推理（Chain-of-Thought Reasoning）
- 指令遵循（Instruction Following）

## 2.5 大语言模型的应用

大语言模型已成为AI应用的基础设施：

- **代码生成**：GitHub Copilot、CodeGeeX
- **内容创作**：文案写作、代码注释生成
- **智能助手**：ChatGPT、Claude、文心一言
- **知识问答**：基于检索增强生成（RAG）的专业问答系统
