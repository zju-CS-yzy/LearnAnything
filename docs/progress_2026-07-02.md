## 三、遗留问题

### P1：连接覆盖率优化
- 当前：68.0%（已达标）
- 可进一步提升至 80%+（方案B：embedding fallback，方案C：prompt 增强）

### P1：数据重复累积
- 多次构建导致数据库中概念节点重复（843 原始 vs 395 canonical）
- 建议：清理数据库，重新完整构建一次

### P2：智谱 API 限流
- embedding 调用被限流（429），降级为 HashEmbedding
- 影响：相似度计算质量下降
- 解决：降低并发或增加重试间隔

### P2：节点卡片优化空间
- 英文换行不够智能（单词被截断）
- 描述过长时节点高度过大
- 间距仍可微调
