### 2. SemanticLinker 0 条边 Bug 修复

**根因分析**：
1. `SemanticLinker` 初始化时没有 `self.embeddings`（`EmbeddingManager` 未初始化）
2. `_get_embedding` 方法不存在，调用时异常被静默捕获，所有 embedding 为 `None`
3. `get_canonical_concepts` 查询只返回 4 个字段，缺少 `parent_hint` 和 `description`
4. `_write_edges` 仍使用旧的 `:Concept` 标签和 `concept_id` 字段

**修复**:
- 初始化 `EmbeddingManager`
- 添加 `_get_embedding` 方法（使用 `embed_single`）
- 批量计算 embedding（`embed()` 一次性计算所有概念）
- 修复 `get_canonical_concepts` 返回完整字段
- 修复 `_write_edges` 使用 `CanonicalConcept`/`canonical_id`
